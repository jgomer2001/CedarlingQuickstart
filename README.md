# CedarlingQuickstart

This repository contains quickstart policy stores for unsigned and multi issuer authorization via Cedarling.

[Cedarling Quick Start Guide](https://docs.jans.io/head/cedarling/quick-start/cedarling-quick-start/)

## Policies in this repository

### Unsigned authorization
- `allow_teacher`

### Signed authorization
- `admin_can_write_to_application`
- `allow_student_read`
- `allow_teacher_secretdocument`
- `cannot_write_if_not_admin`
- `editor_can_share`
- `security_admin_can_write_to_secretdocument`
- `user_can_compare`
- `user_can_execute`
- `user_can_monitor`
- `user_cannot_execute_on_public_network`
- `user_cannot_read_if_compromised`
- `user_cannot_read_if_fraudulent`
- `user_cannot_write_from_restricted_country`
- `user_can_read`
- `user_can_read_if_confidential_role`
- `user_can_read_if_email_is_example`
- `user_can_read_in_corp_network`
- `user_can_search`
- `user_can_tag`
